이벤트 속성 종류
1. 문서(Document) 로딩 이벤트
- load : 개발자가 작성한 문서를 모두 로딩하면 실행하도록 하는 이벤트 종류
- resize : 디바이스 영향 > 문서 화면 크기가 변경되었을 때 사용하는 이벤트 종류

2. 마우스(mouse) 이벤트
- click : 버튼 혹은 특정요소들을 클릭했을 때 실행되도록 하는 이벤트
- dbclick : 버튼 혹은 특정요소들을 더블 클릭했을 때 실행되도록 하는 이벤트
- mouseover : 특정요소 위에 마우스를 올렸을 때 (*버블링 기본적용)
- mouseout : 특정요소에서 마우스를 떠나게 했을 때 (*버블링 기본적용)
- mouseenter : 특정요소 위에 마우스를 올렸을 때 (*버블링 적용하지 않는다)
- mouseleave : 특정요소에서 마우스를 떠나게 했을 때 (*버블링 적용하지 않는다)

3. 키보드(keyboard) 이벤트
- keydown : 사용자가 키를 누르고 있는 동안 발생되는 이벤트
- keyup : 키를 눌렀다가 키에서 손을 뗄 때 발생하는 이벤트

4. 폼 이벤트
- change
- focus
- blur
- submit

5. 위치 이벤트
- pageX : 문서(코드를 작성한 작업데이터)를 기준으로 x축으로 부터 얼만큼 이동한 곳인지 확인
- pageY : 문서(코드를 작성한 작업데이터)를 기준으로 y축으로 부터 얼만큼 이동한 곳인지 확인
- screenX : 현재 화면을 기준으로 x축으로부터 얼만큼 이동한 곳인지 확인
- screenY : 현재 화면을 기준으로 y축으로부터 얼만큼 이동한 곳인지 확인
