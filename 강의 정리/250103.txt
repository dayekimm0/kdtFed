CSS

>html에 css속성을 적용하는 3가지 방법 3가지 방법
-인라인 스타일
-내부 스타일

1) 선택자

* 전체선택자  : *
    html문서 내 모든 태그를 전체 찾아와서 스타일을 적용하고자 할 때
* 태그선택자 : html을 통해서 입력한 태그명을 그대로 가져와서 스타일을 적용할 때
* 아이디 선택자 : html 문서상에 id라는 속성을 부여하고 해당 id 속성값을 활용해서 선택자로 활용하는 것
* 클래스 선택자 : html 문서 상에 class라는 속성을 부여하고 해당 class속성값을 활용해서 선택자로 사용하는 것

//class는 . //id 선택자는 #
&는 위에서 작성한 태그를 한번 더 사용

* 속성선택자 : html 태그 내 속성을 구분하여 스타일 선택자로 사용하는 것
* 후손 & 자손 선택자 : 후손은 자손을 포함한 모든 자식 요소들을 의미 / 자손은 부모 요소의 직계 자식 만을 의미 / 자손은 후손 안에 포함되는 개념
* 동위 선택자 : 동일한 부모 밑에 복수의 형제가 존재할 때 특정 형제를 기준으로 다음번 혹은 복수의 형제를 선택자로 찾아오고 싶을 때 사용
    1) 특정 형제의 바로 다음 번째 형제를 찾아올 때 #ID + li
    2) 특정 형제 아래 모든 형제들을 찾아올 때 #ID ~ li
* 반응 선택자 : 사용자의 이벤트(사용자가 특정 요소에 주는 행위)로 인해서 태그가 반응할 때 스타일 속성을 줄 수 있는 선택자
* 상태 선택자 : html 태그 안에 상태를 정의할 수 있는 태그들이 존재. 해당 태그들의 상태에 따라서 스타일 속성을 상이하게 적용하고자 할 때
    >form>checkbox & radio
    >form>input
* 구조 선택자 : 
    1) 일반 구조 선택자 : 특정 부모 밑에 자식 요소들이 모두 같은 경우에 사용 가능
    first-child
    last-child
    nth-child(2n // 2n+1)
    nth-child(even // odd)
    
    2) 형태 구조 선택자 : 특정 부모 밑에 자식 요소들이 하나라도 상이한 경우 사용 가능
    first-of-type
    last-of-type
    nth-of-type(2n // 2n+1)
    nth-of-type(even // odd)
* 문자 선택자 
    1) 전(前) 문자 선택자 : 특정 텍스트 혹은 태그 요소를 작성한 후에 해당 요소 앞 쪽(좌/상대적 앞쪽)에 어떤 문자 혹은 태그 등을 만들어서 삽입하고자 싶을 때
    
    2) 후(後) 문자 선택자 : 특정 텍스트 혹은 태그 요소를 작성한 후 해당 요소 뒤쪽(우/상대적 뒤쪽=Z축을 기준으로 의미))에 어떤 문자 혹은 태그 등을 만들어서 삽입하고자 할 때
    
    3) 전후문자선택자
    - ::before
    - ::after
    >반드시 내부에 content "" 속성 존재 (없으면 작동하지 않음)


※라이브러리 & 프레임워크
 : 기존 문법을 보다 쉽고, 빠르게 사용할 수 있도록 하기 위해 태어난 요소들
>라이브러리(*도서관) : 기존에 존재하는 언어의 문법 체계를 토대로 그 위에 본인들의 커스텀 문법을 얹힌 상태
>프레임워크(*액자) : 완벽하게 본인들만의 문법 체계를 구축해서 본인들의 시스템을 개발자들이 별도로 학습
ex) Bootstrap, Next.js, Nest.js
